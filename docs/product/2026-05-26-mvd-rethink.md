# MVD Rethink

## New Purpose

Macro Valuation Desk is no longer being developed as a traditional product.

The site is now a personal public research archive whose job is to demonstrate:

- market reasoning
- data analysis capability
- data engineering capability
- the ability to build updating analysis pipelines

The value is in showing how analysis is formed and maintained, not in pretending to be a broad commercial platform.

## Macro Direction

The macro side should move away from permanent pre-defined driver families.

Instead, `Macro` becomes an archive of authored analyses:

- each analysis starts with explicit reasoning
- the reasoning can be supported by data analysis
- each analysis can have its own data pipeline and refresh cadence
- new analyses are added over time only when there is a real research question worth publishing

This means the macro section should feel closer to a research notebook or analysis library than a dashboard taxonomy.

## Equity Direction

The equity side keeps its current basic philosophy.

`Equity Markets` remains a standardized coverage area where each major index follows the same analysis template, the same core datasource logic, and the same comparable valuation frame.

The macro side is flexible and authored.
The equity side is standardized and repeatable.

That contrast is intentional.

## Site Language

All visible site text should reflect the new positioning:

- less product language
- less "desk" language
- more personal authorship
- more emphasis on analysis, reasoning, and pipelines

The site should read like a serious personal research portfolio, not like a startup landing page.

## Implementation Implications

The current cleanup direction is:

1. remove macro placeholders and fixed driver-family framing
2. present macro as an analysis archive
3. keep equity coverage standardized
4. redesign the equity overview into a clearer table-like comparison
5. keep expanding by publishing new analyses and wiring them to refreshable data flows
